Machine Learning & AI — the whole field

  • Machine Learning

    by learning paradigm
    • Supervised Learning trains on labeled examples (input → known output) ▸
      • Regression

        predicts a continuous number, e.g. price, temperature

        Linear Regression Polynomial Regression Ridge Regression Lasso Regression Elastic Net Decision Tree Regressor Random Forest Regressor Gradient Boosting (XGBoost / LightGBM / CatBoost) Support Vector Regression (SVR) KNN Regression Neural Network Regression
      • Classification

        predicts a discrete category, e.g. spam / not spam

        Logistic Regression Decision Tree Random Forest Naive Bayes K-Nearest Neighbors (KNN) Support Vector Machine (SVM) Gradient Boosting (XGBoost / LightGBM / CatBoost) Neural Network Classifier
    • Unsupervised Learning finds structure in data with no labels at all ▸
      • Clustering

        groups similar data points together

        K-Means Hierarchical Clustering DBSCAN Mean Shift Gaussian Mixture Model (GMM)
      • Dimensionality Reduction

        compresses many features into fewer, denser ones

        PCA t-SNE UMAP Autoencoders Linear Discriminant Analysis (LDA)
      • Association Rule Learning

        finds items that tend to co-occur, e.g. market-basket analysis

        Apriori FP-Growth Eclat
    • Semi-Supervised Learning a small labeled set plus a much larger unlabeled set ▸

      used when labeling everything is too expensive, e.g. labeling 5% of medical scans

      Self-Training Pseudo-Labeling Label Propagation Co-Training
    • Reinforcement Learning learns by trial and error, guided by reward signals ▸
      • Value-Based

        learns how good each action is, then picks the best one

        Q-Learning SARSA Deep Q-Network (DQN)
      • Policy-Based

        learns the action-picking strategy directly

        REINFORCE Policy Gradient
      • Actor-Critic

        learns a policy and a value estimate together

        A2C A3C PPO DDPG SAC

  • Deep Learning & Neural Networks

    by architecture
    • Feedforward Network (MLP) the basic layered network — good for tabular data ▸
      Perceptron Multilayer Perceptron (MLP)
    • Convolutional Network (CNN) for images and other spatial / grid data ▸
      LeNet AlexNet VGG ResNet Inception EfficientNet
    • Recurrent Network (RNN) for sequences, where order matters ▸
      Vanilla RNN LSTM GRU
    • Transformer attention-based — behind today's large models ▸
      BERT (encoder-only) GPT (decoder-only) T5 (encoder-decoder) Vision Transformer (ViT)
    • Generative Adversarial Network (GAN) a generator and a discriminator trained against each other ▸
      DCGAN CycleGAN StyleGAN
    • Autoencoder compresses input, then reconstructs it ▸
      Vanilla Autoencoder Variational Autoencoder (VAE) Denoising Autoencoder
    • Diffusion Model generates by learning to reverse a noising process ▸
      DDPM Stable Diffusion Imagen
    • Graph Neural Network (GNN) for data shaped like a graph — social networks, molecules ▸
      GCN GAT GraphSAGE

  • Statistics & Experimentation

    a separate branch, not a part of ML

    ML asks "can I predict this." Statistics asks "is this difference real, and how confident should I be." They share some math but aim at different questions — that's why this sits beside ML rather than inside it.

    • Experimental Design / A-B Testing testing whether a change actually causes a measured difference ▸
      • Core Significance Tests

        checks whether an observed difference is unlikely to be due to chance

        A/B Testing Hypothesis Testing t-test Chi-Square Test ANOVA Mann-Whitney U Test
      • Planning & Validity

        decides how the experiment should be run before it starts

        Power Analysis Sample Size Calculation Randomization Multiple Testing Correction (Bonferroni / FDR)
      • Sequential & Adaptive Methods

        lets the test adapt or stop early as data comes in

        Sequential Testing Bayesian A/B Testing Multi-Armed Bandits
    • Inferential Statistics quantifying uncertainty from a sample about the whole population ▸
      • Estimation
        Confidence Intervals Maximum Likelihood Estimation (MLE) Point Estimation Method of Moments
      • Resampling

        rebuilds the sampling distribution by reusing the data itself

        Bootstrapping Jackknife Permutation Tests
      • Bayesian Inference

        updates a belief about a parameter as new evidence arrives

        Bayes' Theorem Prior / Posterior Updating Markov Chain Monte Carlo (MCMC) Bayesian Credible Intervals
    • Causal Inference estimating cause-and-effect, not just correlation ▸
      • Experimental

        the gold standard — random assignment removes most confounding by design

        Randomized Controlled Trials (RCTs)
      • Quasi-Experimental

        estimates a causal effect when you can't randomize

        Propensity Score Matching Difference-in-Differences Regression Discontinuity Instrumental Variables Synthetic Control
      • Causal Modeling
        Causal Graphs / DAGs Structural Causal Models Do-Calculus Mediation Analysis
